    表格简单说明:
优先看最新日期的表格(看文件名即可)

qlib 预测的目标分数score含义
基于 T 日收盘数据，预测 T+1 日收盘买入 到 T+2 日收盘卖出 的期望收益率。

目前准备了8个模型, 8个模型同时预测, 然后会得到8个预测结果, 最后取平均值作为最终的预测分数。


以下图表格为例

instrument  avg_score   pos_ratio   real_label  datetime         ... (alpha158因子)
SH600115    0.208188    1             nan       2026-02-11
SH600010    0.180931    0.875         nan       2026-02-11


avg_score: 8个模型预测的平均分数, 由label公式可知: 是期望收益率, 没有考虑a股的10%涨停跌停规则.
pos_ratio: 预测结果中正样本的比例, 如果是1表示8个模型全部预测正收益, 如果是0.875表示8个模型预测正收益的有7个, 负收益的有1个
real_label: 真实的label的数值, 这里是nan, 表示没有真实的label, 因为预测的结果是未来的收益率, 未来的收益率是未知的, 
            如果预测日期选择过去的某个交易日, 可以得到真实的收益率, 可以和预测score做个简单的对比
datetime: 预测日期, 表示的含义是 02-11收盘的数据来做预测, 02-12收盘买入, 02-13收盘卖出的期望收益率


score 和 label 是同一个概念, 预测出来收益的一般是score, 真实的收益率是label

文件说明:
total.csv: 模型预测原始数据, 有多个score表格
xxx_ret: 经过计算平均值后的数据, 一个表格
xxx_filter_ret: 经过某个规则过滤后的平均值数据, 一个表格

建议优先观看 xxx_ret表格
