# 关于

## 项目简介

本项目通过 GitHub Actions 实现**每日自动化 CI 流程**：每晚 10 点触发，自动从 [chenditc/investment_data](https://github.com/chenditc/investment_data) 拉取最新 A 股数据，完成模型训练与选股预测。

## 训练流程

1. **数据获取**：自动拉取最新 A 股行情数据
2. **模型训练**：采用仅需 CPU 即可训练的模型（XGBoost、LightGBM、DoubleEnsemble、Linear），分别使用过去 1、2、3、4、5 年的历史数据训练，共得到 20 个模型
3. **模型筛选**：依据 IC、ICIR、Rank IC、Rank ICIR 等指标，筛选出表现较好的模型
4. **预测输出**：多个模型同时预测，取平均分数作为最终预测结果

## 分数含义

**预测分数（score）**：基于 T 日收盘数据，预测「T+1 日收盘买入、T+2 日收盘卖出」的期望收益率。

> 注：该收益率为理论值，未考虑 A 股 10% 涨跌停板限制。  
> **score** 与 **label** 为同一概念：预测值为 score，真实收益率为 label。

## 数据表格说明

**查看建议**：优先查看最新日期的表格（看文件名中的日期即可），建议优先观看 `xxx_ret` 表格。

### 表格列说明

| 列名 | 含义 |
|------|------|
| instrument | 股票代码 |
| avg_score | 多模型预测的平均分数，即期望收益率 |
| pos_ratio | 预测为正收益的模型占比。如 1 表示全部预测正收益，0.875 表示 8 个模型中有 7 个预测正收益 |
| real_label | 真实收益率。预测未来时为 nan；若预测历史日期，可得到真实值用于对比 |
| datetime | 预测日期。如 2026-02-11 表示：基于 02-11 收盘数据预测，02-12 收盘买入、02-13 收盘卖出的期望收益率 |

### 文件类型

| 文件 | 说明 |
|------|------|
| total.csv | 模型预测原始数据，包含多个 score 表格 |
| xxx_ret | 经计算平均值后的汇总数据 |
| xxx_filter_ret | 经规则过滤后的平均值数据 |
